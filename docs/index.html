📄 MCP Server Convert

MCP Server Python AI Agent Tool

Convert PDF, DOCX, HTML, EPUB, CSV, JSON and more to Markdown — designed for AI agents using the Model Context Protocol.

⭐ View on GitHub

Why MCP Server Convert?

AI agents (Claude, GPT, etc.) need to read documents. But most documents aren't in a format LLMs understand well. This MCP server converts any document to clean Markdown that AI agents can process natively.

📄 7+ Formats

PDF, DOCX, HTML, EPUB, CSV, JSON, images (OCR)

🔧 6 MCP Tools

convert_file, convert_url, batch_convert, and more

🐍 Python Native

Zero heavy dependencies. Uses Python standard library

🐳 Docker Ready

Single Dockerfile, one command deploy

Quick Start

# Install
pip install -r requirements.txt

# Run
python -m mcp_server_convert

Configure in Claude Code

{
  "mcpServers": {
    "convert": {
      "command": "python",
      "args": ["-m", "mcp_server_convert"],
      "cwd": "/path/to/mcp-server-convert"
    }
  }
}

Configure with Docker

{
  "mcpServers": {
    "convert": {
      "command": "docker",
      "args": ["run", "-i", "--rm", "-v", "/path/to/files:/data", "mcp-server-convert"]
    }
  }
}
